# lovro_os
Simple operating system in C++

## Prerequisites
Install the following programs:  

```
sudo apt-get install virtualbox
sudo apt-get install grub-legacy-ec2 
sudo apt-get install xorriso 
```

## How to build

```
./CompileAndLink.sh
./GenerateIso.sh
```
